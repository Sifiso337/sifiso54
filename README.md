## Quick Start

1. Clone this repository:
   ```bash
   git clone https://github.com/Sifiso337/sifiso54.git
   cd sifiso54
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the agent:
   ```bash
   python agent.py
   ```

## Configuration
